String methods

String.at()

const str = 'Hello World'

str.at(1) // => 'e'
str.at(-2) // => 'l'
str.at(99) // => undefined

The at() method takes an integer value and returns a new String consisting of the single UTF-16 code unit located at the specified offset. This method allows for positive and negative integers. Negative integers count back from the last string character.

Syntax: at(index) / Return value: String

String.charAt()

const str = 'Hello World'

str.charAt(1) // => 'e'
str.charAt(99) // => ''

The charAt() method of String values returns a new string consisting of the single UTF-16 code unit located at the specified offset into the string.

Syntax: charAt(index) / Return value: String

String.concat()

const str1 = 'Hello'
const str2 = 'World'

str1.concat(' ', str2) // => 'Hello World'

The concat() method concatenates the string arguments to the calling string and returns a new string.

Syntax: concat(string) concat(string1, string2, /* ..., */ stringN) / Return value: String

String.endsWith()

const str = 'Hello World'

str.endsWith('World') // => true
str.endsWith('World', 5) // => false

The endsWith() method determines whether a string ends with the characters of a specified string, returning true or false as appropriate.

Syntax: endsWith(searchString) endsWith(searchString, endPosition) / Return value: Boolean

String.includes()

const str = 'Hello World'

str.includes('Hello') // => true
str.includes('Hello', 5) // => false

The includes() method performs a case-sensitive search to determine whether one string may be found within another string, returning true or false as appropriate.

Syntax: includes(searchString) includes(searchString, startPosition) / Return value: Boolean

String.indexOf()

const str = 'Hello World'

str.indexOf('lo') // => 3
str.indexOf('lo', 5) // => -1

The indexOf() method of String values searches this string and returns the index of the first occurrence of the specified substring. It takes an optional starting position and returns the first occurrence of the specified substring at an index greater than or equal to the specified number.

Syntax: indexOf(searchString) indexOf(searchString, startPosition) / Return value: Found index as Integer, or -1 if not found

String.lastIndexOf()

const str = 'My dog is not like other dogs.'

str.lastIndexOf('dog') // => 24
str.lastIndexOf('dog', 15) // => 3

The lastIndexOf() method of String values searches this string and returns the index of the last occurrence of the specified substring. It takes an optional starting position and returns the last occurrence of the specified substring at an index less than or equal to the specified number.

Syntax: lastIndexOf(searchString) lastIndexOf(searchString, endPosition) / Return value: Found index, or -1 if not found

String.match()

const str = 'Hello World'

str.match(/[A-Z]/g) // => ['H', 'W']
str.match(/[A-Z]/) // => ['H']
str.match(/[0-9]/) // => null

The match() method retrieves the result of matching a string against a regular expression (regex).

Syntax: match(regex) / Return value: Array

String.padEnd()

const str = 'abc'

str.padEnd(8) // => 'abc     '
str.padEnd(8, '.') // => 'abc.....'

The padEnd() method pads the current string with a given string (repeated, if needed) so that the resulting string reaches a given length. The padding is applied from the end of the current string.

Syntax: padEnd(targetLength) padEnd(targetLength, padString) / Return value: String

String.padStart()

const last4DigitsOfCreditCard = '6789'

last4DigitsOfCreditCard.padStart(16, '*') // => '************6789'

The padStart() method pads the current string with another string (multiple times, if needed) until the resulting string reaches the given length. The padding is applied from the start of the current string.

Syntax: padStart(targetLength) padStart(targetLength, padString) / Return value: String

String.repeat()

const str = 'Hello World! '

str.repeat(2) // => 'Hello World! Hello World! '

The repeat() method constructs and returns a new string which contains the specified number of copies of the string on which it was called, concatenated together.

Syntax: repeat(count) / Return value: String

String.replace()

const str = 'My dog is not like other dogs.'

str.replace('dog', 'cat') // => 'My cat is not like other dogs.'
str.replace(/dog/g, 'cat') // => 'My cat is not like other cats.'

The replace() method returns a new string with one, some, or all matches of a pattern replaced by a replacement. The pattern can be a string or a RegEx, and the replacement can be a string or a function called for each match. If pattern is a string, only the first occurrence will be replaced. The original string is left unchanged.

Syntax: replace(pattern, replacement) / Return value: String

String.replaceAll()

const str = 'My dog is not like other dogs.'

str.replaceAll('dog', 'cat') // => 'My cat is not like other cats.'

The replaceAll() method returns a new string with all matches of a pattern replaced by a replacement. The pattern can be a string or a RegExp, and the replacement can be a string or a function to be called for each match. The original string is left unchanged.

Syntax: replaceAll(pattern, replacement) / Return value: String

String.search()

const str = 'Hello World'

str.search(/W/g) // => 6

The search() method executes a search for a match between a regular expression and this String object.

Syntax: search(regex) / Return value: Index of the first match, or -1 if no match was found

String.slice()

const str = 'My dog is not like other dogs.'

str.slice(14) // => 'like other dogs.'
str.slice(3, 6) // => 'dog'
str.slice(-5) // => 'dogs.'
str.slice(-11, -6) // => 'other'

The slice() method extracts a section of a string and returns it as a new string, without modifying the original string.

Syntax: slice(indexStart) slice(indexStart, indexEnd) / Return value: String

String.split()

const str = 'My dog is not like other dogs.'

str.split(' ') // => ['My', 'dog', 'is', 'not', 'like', 'other', 'dogs.']
str.split(' ', 3) // => ['My', 'dog', 'is']
str.split('')[1] // => 'y'

The split() method takes a pattern and divides a String into an ordered list of substrings by searching for the pattern, puts these substrings into an array, and returns the array.

Syntax: split(separator) split(separator, limit) / Return value: Array

String.startsWith()

const str = 'Hello World'

str.startsWith('He') // => true
str.startsWith('He', 3) // => false

The startsWith() method determines whether a string begins with the characters of a specified string, returning true or false as appropriate.

Syntax: startsWith(searchString) startsWith(searchString, startPosition) / Return value: Boolean

String.substring()

const str = 'Hello World'

str.substring(6) // => 'World'
str.substring(6, 9) // => 'Wor'

The substring() method returns the part of the string from the start index up to and excluding the end index, or to the end of the string if no end index is supplied.

Syntax: substring(indexStart) substring(indexStart, indexEnd) / Return value: String

String.toLowerCase()

const str = 'Hello World'

str.toLowerCase() // => 'hello world'

The toLowerCase() method returns the calling string value converted to lower case.

Syntax: toLowerCase() / Return value: String

String.toUpperCase()

const str = 'Hello World'

str.toUpperCase() // => 'HELLO WORLD'

The toUpperCase() method returns the calling string value converted to uppercase.

Syntax: toUpperCase() / Return value: String

String.trim()

const str = '   Hello World   '

str.trim() // => 'Hello World'

The trim() method removes whitespace from both ends of a string and returns a new string, without modifying the original string.

Syntax: trim() / Return value: String

String.trimEnd()

const str = '   Hello World   '

str.trimEnd() // => '   Hello World'

The trimEnd() method removes whitespace from the end of a string and returns a new string, without modifying the original string. trimRight() is an alias of this method.

Syntax: trimEnd() / Return value: String

String.trimStart()

const str = '   Hello World   '

str.trimStart() // => 'Hello World   '

The trimStart() method removes whitespace from the beginning of a string and returns a new string, without modifying the original string. trimLeft() is an alias of this method.

Syntax: trimStart() / Return value: String

Array methods

Array.at()

const arr = [1, 2, 3, 4]

arr.at(1) // => 2
arr.at(-1) // => 4

The at() method takes an integer value and returns the item at that index, allowing for positive and negative integers. Negative integers count back from the last item in the array.

Syntax: at(index) / Return value: Array element

Array.concat()

const arr1 = [1, 2, 3]
const arr2 = [4, 5, 6]

arr1.concat(arr2) // => [1, 2, 3, 4, 5, 6]

The concat() method is used to merge two or more arrays. This method does not change the existing arrays, but instead returns a new array.

Syntax: concat(value) concat(value0, value1, /* … ,*/ valueN) / Return value: Array

Array.every()

const arr = [1, 2, 3]
const isNumber = el => typeof el === 'number'

arr.every(isNumber) // => true

The every() method tests whether all elements in the array pass the test implemented by the provided function. It returns a Boolean value.

Syntax: every(callbackFn) / Return value: Boolean

Array.fill()

const arr = new Array(3)

arr.fill('a') // => ['a', 'a', 'a']
arr.fill('b', 1, 2) // => ['a', 'b', 'a']

The fill() method changes all elements in an array to a static value, from a start index (default 0) to an end index (default array.length). It returns the modified array.

Syntax: fill(value) or fill(value, start, end) / Return value: Array

Array.filter()

const arr = [1, 2, 3, 4, 5]
const threshold = el => el > 3

arr.filter(threshold) // => [4, 5]

The filter() method creates a copy of a portion of a given array, filtered down to just the elements from the given array that pass the test implemented by the provided function.

Syntax: filter(callbackFn) / Return value: Array

Array.find()

const arr = [1, 2, 3, 4, 5]
const threshold = el => el > 3

arr.find(threshold) // => 4

The find() method returns the first element in the provided array that satisfies the provided testing function. If no values satisfy the testing function, undefined is returned.

Syntax: find(callbackFn) / Return value: Array element

Array.findIndex()

const arr = [1, 2, 3, 4, 5]
const threshold = el => el > 3

arr.findIndex(threshold) // => 3

The findIndex() method returns the index of the first element in an array that satisfies the provided testing function. If no elements satisfy the testing function, -1 is returned.

Syntax: findIndex(callbackFn) / Return value: Index

Array.findLast()

const arr = [1, 2, 3, 4, 5]
const threshold = el => el > 3

arr.findLast(threshold) // => 5

The findLast() method iterates the array in reverse order and returns the value of the first element that satisfies the provided testing function. If no elements satisfy the testing function, undefined is returned.

Syntax: findLast(callbackFn) / Return value: Array element

Array.findLastIndex()

const arr = [1, 2, 3, 4, 5]
const threshold = el => el > 3

arr.findLastIndex(threshold) // => 4

The findLastIndex() method iterates the array in reverse order and returns the index of the first element that satisfies the provided testing function. If no elements satisfy the testing function, -1 is returned.

Syntax: findLastIndex(callbackFn) / Return value: Index

Array.flat()

const arr1 = [1, 2, [3, 4]]
const arr2 = [1, 2, [[[3, 4]]]]

arr1.flat() // => [1, 2, 3, 4]
arr2.flat(2) // => [1, 2, [3, 4]]

The flat() method creates a new array with all sub-array elements concatenated into it recursively up to the specified depth.

Syntax: flat() flat(depth) / Return value: Array

Array.forEach()

const arr1 = [1, 2, 3]
const arr2 = []

arr1.forEach(el => arr2.push(el * el))

console.log(arr2) // => [1, 4, 9]

The forEach() method executes a provided function once for each array element. forEach() always returns undefined and is not chainable.

Syntax: forEach(callbackFn) / Return value: undefined

Array.from()

Array.from('abc') // => ['a', 'b', 'c']
Array.from([1, 2, 3], el => el * el) // => [1, 4, 9]

The Array.from() static method creates a new, shallow-copied Array instance from an iterable or array-like object.

Syntax: from(arrayLike) from(arrayLike, mapFn) / Return value: Array

Array.includes()

const arr = ['cat', 'dog', 'horse']

arr.includes('cat') // => true
arr.includes('ca') // => false

The includes() method determines whether an array includes a certain value among its entries, returning true or false as appropriate.

Syntax: includes(searchElement) / Return value: Boolean

Array.indexOf()

const arr = ['cat', 'dog', 'horse', 'cat']

arr.indexOf('cat') // => 0
arr.indexOf('cat', 1) // => 3

The indexOf() method returns the first index at which a given element can be found in the array, or -1 if it is not present.

Syntax: indexOf(searchElement) indexOf(searchElement, fromIndex) / Return value: Index

Array.join()

const arr = ['Bike', 'Car', 'Truck']

arr.join() // => 'Bike,Car,Truck'
arr.join('') // => 'BikeCarTruck'
arr.join(' - ') // => 'Bike - Car - Truck'

Syntax: join() join(separator) / Return value: String

Array.lastIndexOf()

const arr = ['cat', 'dog', 'horse', 'cat']

arr.lastIndexOf('cat') // => 3
arr.lastIndexOf('cat', 1) // => 0

The lastIndexOf() method returns the last index at which a given element can be found in the array, or -1 if it is not present. The array is searched backwards, starting at fromIndex.

Syntax: lastIndexOf(searchElement) lastIndexOf(searchElement, fromIndex) / Return value: Index

Array.map()

const arr1 = [1, 2, 3]

const arr2 = arr1.map(el => el * el) // => [1, 4, 9]

The map() method creates a new array populated with the results of calling a provided function on every element in the calling array.

Syntax: map(callbackFn) / Return value:

Array.pop()

const arr = ['cat', 'dog', 'horse']

arr.pop() // => 'horse'
console.log(arr) // => ['cat', 'dog']

The pop() method removes the last element from an array and returns that element. This method changes the length of the array.

Syntax: pop() / Return value: Array element

Array.push()

const arr = ['cat', 'dog', 'horse']

arr.push('sheep') // => 4
console.log(arr) // => ['cat', 'dog','horse', 'sheep']

The push() method adds the specified elements to the end of an array and returns the new length of the array.

Syntax: push(element) push(element0, element1, /* … ,*/ elementN) / Return value: Array length

Array.reduce()

const arr = [1, 2, 3]

arr.reduce((a, b) => a + b) // => 6

The reduce() method executes a user-supplied "reducer" callback function on each element of the array, in order, passing in the return value from the calculation on the preceding element. The final result of running the reducer across all elements of the array is a single value.

Syntax: reduce(callbackFn) / Return value: Result of reducer callback funtion.

Array.shift()

const arr = ['cat', 'dog', 'horse']

arr.shift() // => 'cat'
console.log(arr) // => ['dog', 'horse']

The shift() method removes the first element from an array and returns that removed element. This method changes the length of the array.

Syntax: shift() / Return value: Array element

Array.slice()

const arr = ['cat', 'dog', 'horse']

arr.slice() // => ['cat', 'dog', 'horse']
arr.slice(1) // => ['dog', 'horse']
arr.slice(1, 2) // => ['dog']

The slice() method returns a copy of a portion of an array into a new array object selected from start to end (end not included) where start and end represent the index of items in that array. The original array will not be modified.

Syntax: slice() slice(start) slice(start, end) / Return value: Array

Array.sort()

const arr1 = ['Anna', 'Charlie', 'Ben']
const arr2 = [1, 5, 7, 12, 500]

arr1.sort() // => ['Anna', 'Ben', 'Charlie']
arr2.sort() // => [1, 12, 5, 500, 7]
arr2.sort((a, b) => a - b) // => [1, 5, 7, 12, 500]

The sort() method sorts the elements of an array in place and returns the reference to the same array, now sorted. The default sort order is ascending, built upon converting the elements into strings, then comparing their sequences of UTF-16 code units values.

Syntax: sort() sort(compareFn) / Return value: Array

Array.splice()

const arr1 = ['one', 'two', 'three']
const arr2 = ['one', 'two', 'three']

arr1.splice(1) // => ['two', 'three']
arr2.splice(1, 2, 'four', 'five') // => ['three']

console.log(arr1) // => ['one']
console.log(arr2) // => ['one', 'two', four', 'five']

The splice() method changes the contents of an array by removing or replacing existing elements and/or adding new elements in place.

Syntax: splice(start) splice(start, deleteCount) splice(start, deleteCount, item0, item1, /* … ,*/ itemN) / Return value: Deleted element(s)

Object methods

Object.assign()

const obj1 = { a: 1, b: 2 }
const obj2 = { b: 3, c: 4 }

Object.assign(obj1, obj2) // => { a: 1; b: 3, c: 4 }

The Object.assign() static method copies all properties from one or more source objects to a target object. It returns the modified target object.

Syntax: Object.assign(target, ...sources) / Return value: Object

Object.create()

const obj = { firstname: 'Peter', name: 'Parker' }

Object.create(obj) // => { firstname: 'Peter', name: 'Parker' }

The Object.create() static method creates a new object, using an existing object as the prototype of the newly created object.

Syntax: Object.create(proto) / Return value: Object

Object.entries()

const obj = { firstname: 'Peter', name: 'Parker' }

Object.entries(obj) // => [['firstname': 'Peter'], ['name': 'Parker']]

The Object.entries() static method returns an array of a given object's own enumerable string-keyed property key-value pairs.

Syntax: Object.entries(obj) / Return value: Array

Object.fromEntries()

const arr = [['firstname': 'Peter'], ['name': 'Parker']]

const obj = Object.fromEntries(arr) // => { firstname: 'Peter', name: 'Parker' }

The Object.fromEntries() static method transforms a list of key-value pairs into an object.

Syntax: Object.fromEntries(iterable) / Return value:

Object.keys()

const obj = { firstname: 'Peter', name: 'Parker' }

Object.keys(obj) // => ['firstname', 'name']

The Object.keys() static method returns an array of a given object's own enumerable string-keyed property names.

Syntax: Object.keys(obj) / Return value: Array

Object.values()

const obj = { firstname: 'Peter', name: 'Parker' }

Object.values(obj) // => ['Peter', 'Parker']

The Object.values() static method returns an array of a given object's own enumerable string-keyed property values.

Syntax: Object.values(obj) / Return value: Array
